Сommand list

  • Команда: git --version позволяет проверить , какая сейчас установлена версия GitHub
  • Команда: git init национaлизирует файл , как git проэкт (подразумеват изминение файлов index.html и др. в зеленый цвет)
  • Команда: git status позволяет посмотреть отслеживаются файлы в репазиторий
  • Команда: git add . добавляет файлы в репазиторий
  • Команда: git commit -m Позволяет указать все изменения , которые были внесены в файл под одним коротким описанием
  • Команда: git Log позволяет посмотреть историю своих commit
  • Команда: it remoute -v позволяет проверить , где находятся файлы
